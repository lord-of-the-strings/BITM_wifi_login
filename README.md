# BITM_wifi_login
A simple automation which connects our hostel/campus wifi easily
Quick Steps to Automate the Boring Process of Wifi Login
1) Clone this repository.
2) Add sophos_login.bat to taskschd.msc to schedule task on logon.
3) Change the username and password (line 6 and 7) in hostel_wifi.py and ENJOY!!
In just three steps, you can automate login on any windows machine! (Work in progress for other operating systems!)
